Projeto - 01
Project - 01
Ingressos de cinema
Cinema tickets
Este é um projeto criado por mim simulando a venda de ingressos de cinema para assistir ao filme da Barbie.
This is a project I created that simulates selling movie tickets to watch the Barbie Movie.
Ingressos de cinema
Cinema tickets

O projeto tem quatro páginas web.

Na homepage foi colocado um banner com uma cena do filme, trailler, sinopse e um botão para o usuário comprar o ingresso.

Na segunda página, foi feita a apresentação dos personagens por meio de um slide.

A terceira página é onde o usuário compra o bilhete, escolhendo o endereço do cinema, data, sessão e até o assento que irá utilizar. Simula o pagamento e finaliza. Todo o cadastro foi feito com validação dos valores dos inputs.

Por final, uma quarta página se abre com os dados do bilhete comprado.

The project has four web pages.

The homepage features a banner with a scene from the movie, a trailler, a synopsis, and a button for the user purchase the ticket.

The second page presents the characters through a slideshow.

The third page is where the user purchases the ticket, selecting the cinema location, date, session, and even the seat. It simulates the payment and finalizes the purchase.

Finally, a fourth page opens displaying the details of the purchased tickets.

A simulated Asics sneaker e-commerce platform developed using React.
E-commerce
E-commerce

O projeto tem três etapas.

A homepage, onde são expostos todos os produtos e que podem ser filtrados por categorias (masculinos, femininos e infantil) e preço.

Uma parte dedicada ao carrinho, onde o usuário poderá verificar os itens da compra, remover e adicionar itens e checar o valor total da compra.

Uma segunda página com o formulário para obter os dados do comprador e entrega, além de simular o pagamento do produto.

O rápido preenchimento do endereço teve a ajuda da API ViaCep.

The project consists of three pages.

The homepage displays all products, which can be filtered by category (men's, women's, and children's) and price.

A dedicated section for the shopping cart, where the user can view their items, remove or add items, and check the total purchase amount.

A second page with a form to collect the buyer's and shipping information, as well as a simulated payment process.

The rapid filling of the address was assisted by the ViaCep API.
